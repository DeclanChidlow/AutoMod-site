Welcome to the documentation for the AutoFeeds bot! If you're looking for help using AutoFeeds, you're in the right place!

Chances are you want to [set it up](/docs/autofeeds/setup) and then head over to the [commands page](/docs/autofeeds/commands) to learn all the various things you can do with the bot.

If you're hosting your own Stoat instance, or are otherwise interested in hosting your own version of AutoFeeds, you'll want the [self-hosting](/docs/autofeeds/self-hosting) page.

If you need some help using AutoFeeds, you can get some over in the support channel on [the AutoMod Stoat server](https://stt.gg/automod).

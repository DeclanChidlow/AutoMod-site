Welcome to the documentation for the AutoMod bot! If you're looking for help using AutoMod, you're in the right place!

Chances are you want to [set it up](/docs/automod/setup) and then head over to the [commands page](/docs/automod/commands) to learn all the various things you can do with the bot.

If you're hosting your own Stoat instance, or are otherwise interested in hosting your own version of AutoMod, you'll want the [self-hosting](/docs/automod/self-hosting) page.

If you need some help using AutoMod, you can get some over in the support channel on [the AutoMod Stoat server](https://stt.gg/automod).

Permit users or roles to bypass AutoMod's moderation rules.

Aliases: `whitelist`, `safelist`

## Add/Remove User/Role

Adds or removes a user or role from the allowlist.

Usage: `/allowlist <add|remove> <user|role>`

## List

AutoMod will post a list of allowlisted users and roles.

Usage: `/allowlist list`
